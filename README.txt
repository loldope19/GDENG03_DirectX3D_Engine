How to Run
-----------------------------------------
This project is intended to be built and run using Visual Studio 2022.

1. Open the 'GDENG03_DirectX3D_Triangle.sln' solution file in Visual Studio.
2. Make sure the startup project is set to 'GDENG03_DirectXGame'.
3. Press the "Local Windows Debugger" button (with the green play icon) or press the F5 key on your keyboard. This will build and run the program.


Controls
-----------------------------------------
- Right-Click + Mouse Move: Look around with the camera.
- Right-Click + WASD Keys: Move the camera forward, left, backward, and right.
- Spacebar: Toggle the rotation of the cubes.
- +/- Keys: Adjust the engine's time scale (slow-motion/fast-forward).
